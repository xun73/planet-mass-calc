# 🪐 Planet Mass Calculator

A browser-based tool for measuring satellite orbits from astrophotography and calculating planet mass using Kepler's Third Law.

[https://xun73.github.io/planet-mass-calc/](https://xun73.github.io/planet-mass-calc/)

## Features

- Upload astronomical images, mark satellites, and fit orbits
- Lomb-Scargle periodogram for period analysis
- Auto-fit algorithm with brute-force grid search
- Mass calculation via Kepler's Third Law
- English / Chinese bilingual interface (auto-detects browser language)

## Usage

1. Upload images (drag & drop or batch select)
2. For each image: set observation time, mark primary mask and satellites
3. Switch to "Orbit Fitting & Mass Calculation" tab
4. Use auto-fit or manually adjust parameters
5. View mass results in the calculation table

# 🪐 行星質量計算器

一套瀏覽器端的天體軌道測量與質量計算工具，透過克卜勒第三定律計算行星質量。

[https://xun73.github.io/planet-mass-calc/](https://xun73.github.io/planet-mass-calc/)

## 功能

- 上傳天文照片、標記衛星並進行軌道擬合
- Lomb-Scargle 週期圖分析
- 暴力搜尋自動擬合演算法
- 克卜勒第三定律質量計算
- 中 / 英文雙語介面（自動偵測瀏覽器語言）

## 使用方式

1. 上傳照片（拖曳或批次選擇）
2. 每張照片：設定拍攝時間、標記主星遮罩與衛星
3. 切換至「軌道擬合與質量計算」頁籤
4. 使用自動擬合或手動調整參數
5. 檢視計算表格中的質量結果
